READ ME (dataset names)

observations are at the county-majorindustry level

state - state FIPS name
state_code - state FIPS code
county_code - county FIPS code
fips - full FIPS code (i.e., state and county)
major_industry_of_parent - Subsidy Tracker's description of major industry
naics - six-digit NAICs classification code
tsubs - total subsidies received by a particular county-majorindustry between 2000 and 2019
